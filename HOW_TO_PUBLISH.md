# MathDefy Showcase Repo

This folder is a ready-to-publish public showcase version of the project.

## What to do next

1. Create a new public GitHub repository named `MathDefy-showcase`
2. Copy the contents of this folder into that repository
3. Add your screenshots into `docs/screenshots/`
4. Commit and push

## Included here

- `README.md`: public showcase presentation
- `LICENSE`: noncommercial license
- `docs/screenshots/`: screenshot folder ready to use

## Important

Do not copy source code files from the private project if you want to keep the implementation private.
